Documentație Proiect Web – I rate it!

Sebastian Popa, Dascaliu Ianis

Rezumat

Acest proiect web permite gestionarea evaluărilor și părerilor exprimate de utilizatori cu privire la o entitate X, unde X poate fi o categorie sau un element individual dintr-o clasă de interes: profesie, instituție, companie, produs sau serviciu. Utilizatorii pot adăuga recenzii, pot comenta, pot încărca imagini, iar sistemul generează statistici, inventare și clasamente, cu posibilitate de export în formate deschise (CSV, PDF) și flux RSS.

Arhitectură și tehnologii

  • Backend: Node.js (router custom), PostgreSQL
  • Frontend: HTML, CSS, JavaScript (UI custom)
  • Cloud: AWS EC2 (hostare), AWS S3 (stocare imagini)
  • Export: CSV (server-side), PDF (client-side, jsPDF)
  • Altele: RSS 2.0 pentru clasamente

Funcționalități principale

  • Autentificare și înregistrare utilizatori (JWT, cookie-uri securizate)
  • Adăugare, vizualizare, filtrare și ștergere recenzii pentru entități X
  • Comentarii la recenzii, cu rating și imagini
  • Inventar și statistici: totaluri, medii pe categorii, top utilizatori
  • Clasament entități dezirabile/detestabile (top/flop) – vizual și RSS
  • Export inventar recenzii în CSV (server-side) și PDF (client-side)
  • Import date din CSV/JSON
  • Administrare utilizatori, recenzii, bug reports (interfață admin dedicată)

Model de date și caracteristici recenzie

  • Entitate: Numele subiectului evaluat (ex: "Liceul X", "iPhone 13")
  • Categorie: Domeniul sau clasa (ex: "liceu", "telefon")
  • Rating: Notă între 1 și 5
  • Comentariu: Text justificativ, cu minim o trăsătură (in)dezirabilă
  • Imagini: Opțional, maxim 3 per recenzie/comentariu (stocate în S3)
  • Comentarii la recenzii: Fiecare cu rating opțional și imagini

Export și import date

Export CSV

Exportul CSV se realizează server-side și conține: id, entitate, categorie, medie rating, comentariu, autor.
Exemplu de antet CSV: id,entity,category,avg_rating,comment,username

Export PDF

Exportul PDF se realizează client-side cu jsPDF și include tabelul recenziilor vizibile, cu aceleași coloane ca la CSV.

Import CSV/JSON

Utilizatorii pot importa recenzii din fișiere CSV sau JSON, cu validare automată a câmpurilor și a ratingului.

Clasament și flux RSS

Sistemul calculează automat top 5 cele mai apreciate și cele mai detestate entități (minim 3 review-uri), pe baza mediei ratingurilor din recenzii și comentarii. Clasamentul este disponibil atât vizual, cât și ca flux RSS 2.0 la /clasament.rss.

Exemplu item RSS

<item>
  <title>Top 5 Dezirabile</title>
  <description><ol>
    <li><span>Categorie</span> — <b>Entitate</b>: 4.75/5 (10 review-uri)</li>
    ...
  </ol></description>
</item>

Interfață utilizator

  • Pagina principală: Vizualizare, filtrare, sortare, adăugare recenzii, export/import, statistici, clasament, RSS
  • Autentificare/înregistrare: Formulare simple, cu validare și feedback
  • Admin: Gestionare utilizatori, recenzii, bug reports

Fluxuri și API-uri principale

  • POST /register, POST /login, POST /logout – autentificare
  • GET /get-reviews – listare recenzii
  • POST /add-review – adăugare recenzie
  • GET /export-csv – export inventar CSV
  • GET /clasament.rss – flux RSS clasament
  • POST /import-csv, POST /import-json – import date
  • GET /get-my-reviews, GET /get-my-comments – recenzii/comentarii proprii
  • POST /report-bug – raportare bug

Considerații de securitate și scalabilitate

  • Stocare imagini în AWS S3, acces securizat
  • Autentificare JWT, cookie-uri HttpOnly, SameSite
  • Validare date la import/export
  • Scalabilitate: backend stateless, poate fi extins pe mai multe instanțe EC2
